#Lists 

When you pass a list to a function, the function gets a reference to the list.
- If the function modifies a list parameter, the caller sees the change.

**delete_head** removes the first element from a list.
```
def delete_head(t)
    del t[0]
```

Heres how its used:
```
letters = ['a', 'b', 'c']
delete_head(letters)
print(letters)
```
