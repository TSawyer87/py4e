- A value is one of the most basic things a program works with, like a letter or a number.
- These values belong to different types: Integer, Floating Point, String, etc.

```python
type('Hello, World!')
<class 'str'>
type(17)
<class 'int'>
type(3.2)
<class 'float'>
type('17')
<class 'str'>
type('3.2')
<class 'str'>

>>> print(1,000,000)
```
